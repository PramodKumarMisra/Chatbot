# Chatbot
#It is chatbot for everyone 
This Chatbot run into Jupyter_Notebook under  Anaconda-platform

**Prerequist:-**

  You need Python(3.7).
  
  Anaconda (Jupyter_Notebook).
  
  Pip library. 
  
  Dependent python packages:-
  
            1. NumPy
            
            2.NLTK
            
            3.random2
            
            4.String
            
            5.sklearn 
  
  Internet Cinnection.
  

 
Install  dependencies by running:-
   
     pip install numpy
     
     pip install nltk
     
     pip install random
     
     pip install string
     
     pip install sklearn
